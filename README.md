A Full Stack Web Application built to manage employee records.
Built with **Java Spring Boot**, **MySQL**, and **Vanilla JavaScript**.

## Features
- **Hire Employees:** Add new records to the database via a Dynamic UI.
- **View Team:** Real-time fetching of employee data from the backend.
- **REST API:** Handles POST and GET requests efficiently.

## Tech Stack
- **Backend:** Java, Spring Boot, Spring Data JPA
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)

## How to Run
1. Clone the repository.
2. Update `application.properties` with your MySQL credentials.
3. Run `DemoApplication.java`.
4. Open `test.html` in your browser.
